How to Download Our Game

Instructions:

1. Click on the button to download a zip file

2. In order to run the game, you must have python and pygame downloaded. Here are the links to download both (make sure versions are compatible with each other):

Python: https://www.python.org/downloads/

Pygame: http://www.pygame.org/download.shtml

3. Once both downloads are complete, go to your 2016_SIP_AdobeSJ_Lovelace folder.

4. (OPTIONAL) If you want, find the time_runner.py file, right click, and click on “Edit with IDLE”. This would open the app that you would play and allow you to see the code for it.

5. Open command prompt in the downloaded folder, and type (without quotes):

“cd 2016_SIP_AdobeSJ_Lovelace_TimeRunner-master” (enter)

“python time_runner.py”
